• Ezzing 3DLayout is a PV planning tool that allows you to generate a 3d model of a building based on a satellite image. You can model any number of buildings, select between up to five different type of roofs, define keepouts and trees with custom heightsâ€¦
  • Inside each roof area you can customize different structures, select module models and get automated previews of your setup.
  • It also provides you with a perspective view and a sun simulator to determine where the shadows will be in your installation.
  • Ezzing 3DLayout is an embebible webapp. You can integrate it inside your own system and customize many elements inside, from module models to prefered default settings for each roof type.
  • You can use this sand box to play with an example of the 3dLayout integrated and see all the functions and examples of this integration system, to get more information about the integration please go to the url https://github.com/ezzing/3DLayout-documentation .
  • To see all the functions, type "help" in the bottom of this console, "start" to start traking the diferent event, "clear" to clear all the message in the console, to init a new project type "newLayout" and to open a showcase view type "showcase".
